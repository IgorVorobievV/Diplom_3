## Задание 3: Автотесты для UI

### Автотесты для проверки UI для Stellar Burgers.

### Реализованные сценарии

Созданы тесты эндпоинтов API для Stellar Burgers.

Созданы отчеты в Allure (отчет: `Diplom_3\allure_results`)

### Структура проекта

- `tests` - пакет, содержащий тесты.
- `pages` - пакет, содержащий вспомогательные функции, разделенные по классам.
- `locators` - пакет, содержащий локаторы, разделенные по классам.
- `data` - пакет, содержащий исходные данные приложения.

### Запуск автотестов

**Установка зависимостей**

> `$ pip install -r requirements.txt`

**Запуск автотестов и отчета**

>  `$ pytest  --alluredir=allure_results`

**Создание HTML-отчета**

>  `$ allure serve allure_results`
